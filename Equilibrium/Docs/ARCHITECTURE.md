## Code Organization Principles

### Single Responsibility

```swift
// Good
class BreathViewModel {
    func startBreathing() { ... }
    func stopBreathing() { ... }
}

// Bad
class BreathViewModel {
    func startBreathing() { ... }
    func saveSettings() { ... }  // Should be in SettingsManager
    func playAudio() { ... }     // Should be in AudioManager
```

### Declarative over Imperative

